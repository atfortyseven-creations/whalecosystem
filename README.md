# Humanity Ledger + Whale Network
ZK Privacy Dashboard & Real-time Scanner built natively on Aztec Network

Proyecto principal de Whale Network (humanidfi.com) enfocado en privacidad programable por defecto.

### Características principales
- Noir circuits para private state (Whale Chat con encrypted logs + Humanity Ledger Scanner)
- Custom Aztec account contracts con QR-code multi-device synchronization
- Non-custodial PWA + mobile apps
- 6.558 contribuciones en 2026 (actividad full-time)
